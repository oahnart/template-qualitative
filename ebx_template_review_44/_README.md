## 발췌 선택 = 항목 고유 적합도 + 섹터 내 중복금지 (2026-06-15 개선)
이전 판은 "문장 깨끗함 + 아무 키워드 매칭"만으로 골라 **한 단락이 여러 항목에 재사용**되고(섹터당 고유
13~19/27) **오주제**가 섞였다(예: ESG비전 칸에 안전보건위 내용). 이번 판은:
1. **항목 고유 적합도(relevance)** — 항목 term 프로파일(키워드 ∪ aspect명 토큰)을 **cross-item idf**로
   가중. 변별 용어(산업안전보건위원회·CISO·ISO 9001·비전·거버넌스 w≈3.3)는 크게, 공유 용어(ESG·안전
   보건·품질 w≈2.0)는 작게, 편재어(체계·관리 w≈1.4)는 거의 0. cleanliness는 0.4 보조항(잡음 필터·동점용).
2. **섹터 내 중복금지** — 항목×문서 후보를 점수순 **그리디 1:1 배정**(출처지문 재사용 금지). 적합 단락이
   남지 않은 항목은 억지 배정 대신 **미확보**(정직한 공백).

→ **섹터별 J 고유 = 채움수(재사용 0)**, 오주제 약 40%대 → **2.4%**로 격감.

## 회사명 마스킹 — 방법·검증·보강 (2026-06-15 갱신: 모델 NER + 적대감사 0 수렴)
- 라벨은 **지주사**인데 본문은 **운영사·시설명**이라 정규식만으론 누출 → **31B-it 익명화**(공공기관·법령
  유지) + 결정론 `mask_orgs`(접미사·시설접미 네트 + 블록리스트)가 1차.
- **⚠ "누출 0" 정정**: 블록리스트만으론 불충분했다. J·K 전수 **적대감사가 블록리스트 밖 잔존 식별토큰
  ~40종을 10/11 섹터에서 적발**(비전·전략 슬로건·영문/사내 약어·프로그램·시설·소재지·그룹이념; 일부는
  웹검증으로 실기업 특정). 가제티어·접미 블록리스트는 **신규 고유명을 원천적으로 못 잡고**, K는 'J복사+수치
  마스킹'이라 J의 고유명사 누출을 상속한다.
- **EBX-NER-MASK 도입(`eval/ebx_remask.py --ner`)** — 3단: ①결정론 `mask_orgs` ②**모델 NER**(31B-it가 가려야 할
  식별 span 목록만 반환→코드가 **원문 span만 결정론 치환=날조0**; keep가드=표준·공공기관·검증기관·수상·범용
  플랫폼·순수기능명) ③**섹터 일관성 + 누적 span 메모리**(`_ner_spans.json`, 단조=이전·감사확정 발견을 영구
  적용) + 마스크 인접 법인접미 흡수.
- **검증**: mask→재감사 **3회 반복으로 잔존 누출 40→7→3→0** 수렴. 최종 **독립 적대감사 확정 누출 0**(11섹터,
  169 고유 span 마스킹). keep 대상(ISO·BSI·환경부·UNGC·KoSIF·일반부서명) 전부 보존(과마스킹 없음). 단
  자동검사·모델판정이 100%를 수학적으로 보장하진 않으므로 외부배포 전 **검수자 최종 눈검수 권장**.
- 재현: `python3 eval/ebx_gen44.py --out <폴더>` (J·K 생성) → `python3 eval/ebx_remask.py --out <폴더> --ner`
  (NER 마스킹; `_ner_spans.json` 있으면 누적 적용). 마스킹 로그=`_ner_mask_log.csv`.

## 검수 시 유의 (정직한 한계)
1. **J는 실데이터 기반**: 타사 실수치 포함(게이트 비대상) → **복사 금지**, 회사명 마스킹 검수자 확인.
2. **주제일치**: on-topic 79.4%·**off-topic 0**(7건 표적 교체로 4 정주제화·3 미확보 강등). borderline 59건
   (주로 6 지표형)은 산문↔지표 구조차 — `_topic_review.csv`로 항목·사유 제공.
3. **마스킹**: 모델 NER + 반복 적대감사로 잔존 누출 40→0 수렴(독립 감사 확정 0). 단 모델판정이 100%를
   수학적으로 보장하진 않으니 외부배포 전 눈검수 권장. 다중 계열사 지주(FN 등) 셀은 마스크가 조밀.
   누출 위험은 **J(동종실제발췌)에 집중** — 절대안전 필요 시 외부배포는 I열만 쓰고 J 제외 고려.
4. **중소 ≡ 비상장**: 내용 동일. 규모 차이=취득 메모 + 지표형 6항목 간소화 지침.
5. 모든 `[ ]`는 회사 작성란. 채운 뒤 grounding 검수(EBX-QUAL-MAP/GEN) 필수.
